Transcribe Tutorial Modal

How to Transcribe this label

Please tell us what the highlighted text says. Try to include any diacritics, like accents ( ` ) or umlauts ( ¨ ), as well as upper and lower case letters.

If something is illegible, that is important information for us to know as well, so please mark it as illegible.

If the area highlighted is an image, please use on or two words to describe the image.
